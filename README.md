# README

### アプリケーション名
Sportweb

--------------------------------------------------------------------------------------------------------------------------------

### 概要
スポーツに関心のあるユーザーとスポーツ活動を行うチームが集まるSNSで、ユーザー同士、ユーザーとチーム、チーム同士の交流が可能。
スポーツ活動を行いたいと考えるユーザーと仲間を募集したいと考えるスポーツチームとのマッチングや、ユーザー同士でのメッセージ交換等を通じて、
スポーツを軸とした繋がりを形成することができる。

--------------------------------------------------------------------------------------------------------------------------------

### コンセプト
スポーツでつながろう
